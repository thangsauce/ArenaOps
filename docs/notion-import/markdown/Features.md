# Features

## Platform Scope

- Tournament operations for E-Sports, sports, board games, and card games
- 25+ games across four categories
- Organizer-focused workflow from creation through live operations

## Landing Page

- Animated hero and scroll-triggered section reveals
- Scoreboard decode animation and per-character flip effects
- Category filtering for E-Sports, Sports, Board, and Card
- Branded game cards with logos and category styling

## Tournament Management

- Create tournaments with configurable format, participant cap, organizer name, and description
- Supported formats include single-elimination, double-elimination, round-robin, swiss, free-for-all, group-stage, battle-royale, and ladder
- Tournament detail view with matches, participants, schedule, and venue data

## Participants

- Registration tracking with confirmed, pending, and declined statuses
- Search, sorting, filtering, and inline status management
- Availability confirmation flow for registered players

## Schedule And Live Ops

- Grid and list schedule views
- Live score entry and match state updates
- Delay reporting, no-show handling, and room reassignment
- Time block and venue-aware scheduling

## Room Booking

- Venue assignment for matches
- Conflict detection for overlapping bookings
- Room availability and usage visibility

## Notifications

- In-app notifications for match and venue events
- Time-grouped notification feed
- Web Audio API sound feedback
- Per-event notification settings and unread indicators

## Settings And UX

- Theme, density, time format, timezone, and notification preferences
- Global sidebar search across app entities
- QR sharing for tournaments
- Accessibility-focused navigation and reduced-motion support
